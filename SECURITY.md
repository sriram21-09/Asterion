# Security Policy

## Supported Versions

We actively support security updates for the following versions of Asterion:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a potential security vulnerability in Asterion, please follow these steps:

1. **Do not** open a public issue.
2. Email your findings to the maintainer's contact email.
3. Describe the vulnerability, its impact, and steps or code to reproduce it.

We will acknowledge your report within 48 hours and work with you to resolve the issue before making a public disclosure.
